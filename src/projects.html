Jaspal’s Voice

HTML, CSS, JavaScript, Java, Play, AWS, Sketch

When Susan’s sister Jaspal contracted Motor Neurone Disease, one of the hardest things was when Jaspal lost her voice to the disease. However Jaspal found innovative ways of communicating despite the condition.

The charity aims to collate and review communication devices for those suffering from MND. We aim to have a catalogue of devices with user-generated reviews and tips. We hope to be able to donate devices to those in need.

Pimp My Cause

PHP, HTML, CSS, JavaScript, Heroku, AWS

Pimp My Cause is an organisation that connects marketers who wish to volunteer to good causes, successfully matchmaking over 1300 charities and volunteers since its inception in 2011.

The vision behind the web rebuild project is to modernise the tech stack, improve UX, and leave Pimp with a slick new website that is robust, sustainable and capable of supporting the growth in its membership expected over the next few years.

IPPR

Express, CartoDb, Leaflet, ejs, jQuery

This project is aiming to improve transparency around oil contracts. For that, we are building a website using NodeJS/Express on the backend and JavaScript on the front end. We are keeping the data on CartoDB, which includes a PostGIS database on the cloud and some data manipulation and analysis tools. We also use LeafLet, which is a JavaScript framework to plot the CartoDB data on a map. We have currently reached our first milestone and work is still on going.
